# auto trade future binance
this tool auto trade future binance and post result to telegram bot, to check result please chat with bot @auto_trade_binace_bot start command is `start` end command is `end`

## install
```shell
npm install
```

## run in debug
```shell
node index.js
```

## start
```shell
pm2 start index.js
```

## stop
```shell
pm2 stop index.js
```
